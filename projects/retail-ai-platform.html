Retail AI Platform

Real-time analytics platform for retail environments using computer vision, LLMs, and AI-powered multi-agent insights.

Problem

Retail stores lack real-time visibility into foot traffic patterns, congestion hotspots, and operational inefficiencies. Manual monitoring is labour-intensive and provides no actionable analytics or predictive insights.

Solution

A comprehensive platform combining YOLOv8 computer vision for real-time people detection, a multi-agent LLM system for intelligent alerting, and an Angular dashboard with live metrics, historical analytics, trend prediction, and staffing recommendations.

Outcome

  • Real-time people counting and congestion detection via YOLOv8 CNN
  • Multi-agent LLM system for coordinated monitoring and intelligent alert generation
  • Live Angular dashboard with 3-second refresh intervals and historical trend analysis
  • Statistical analytics for peak hour detection and staffing optimisation
  • RAG-based knowledge system for contextual alert generation

Technologies

Python Angular PostgreSQL FastAPI YOLOv8 OpenCV Ollama LLM Multi-Agent
