Events / Fundraising

RAISEit

A real-time auction engine with structured fundraising workflows, designed for transparent competitive bidding at scale.

Problem

Auction-based fundraising and team-selection events are typically run manually on spreadsheets or whiteboards. This creates no real-time visibility for participants, high administrative overhead during live rounds, no budget enforcement, no permanent bid record, and no scalable format for multi-team simultaneous events.

Solution

A hosted web platform where admins configure auctions end-to-end (budgets, team limits, player pools, round timers); teams join via unique invitation codes; live WebSocket connections push bid updates to every participant instantly; server-side timers auto-complete rounds; MongoDB persists complete bid history and budget deductions transparently.

Outcome

  • Eliminated manual spreadsheet management for multi-team auction events
  • Server-enforced budget constraints removed human error and dispute possibilities in bidding
  • Auto-complete logic enabled unattended auction closure, reducing admin presence requirement
  • Transparent bid history visible to all participants increased trust in auction outcomes
  • Platform accessible as a live deployed website, enabling use by external organisers
